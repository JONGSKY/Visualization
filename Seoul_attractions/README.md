### 2. 시각화 설명

- 서울 지역 내에서 할만한 데이트 코스를 제공합니다.
- 지역을 선택하면 놀거리들을 지도를 통해 확인가능합니다.
- 모바일 형태로 모바일에서 보다 효과적으로 이용가능합니다.

#### 시각화 만든 이유

- 데이트 코스를 찾는게 힘든데 이를 대시보드로 표현해 좀 더 편한 데이트 코스를 제공하고 싶었습니다.
- 다른 나라 언어들도 포함한 대시보드를 통해 외국인들도 쉽게 관광명소를 볼 수 있도록 하고싶었습니다.
