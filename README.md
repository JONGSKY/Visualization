## 시각화 규칙
1. 적어도 한 달에 한 번 하고싶은 주제에 대해 시각화합니다.
2. 시각화는 의미있고 만든 이유가 있어야 합니다.
3. 시각화결과에 대한 인사이트를 작성합니다.
4. 데이터 출처를 항상 밝히도록 합니다.
